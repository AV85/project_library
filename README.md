# README

HOW TO INSTALL?

1. Clone project. Run ```https://github.com/AV85/project_library.git```
2. Run ```bundle```
3. Rename ```database.yml.example``` to ```database.yml```.
4. Config file ```database.yml```
5. Run ```rake db:create```,  ```rake db:migrate```, ```rake db:seed```.
6. Run ```rails s```
